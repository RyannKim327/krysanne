TeleAI — Compassionate AI Companion

TeleAI (named Krysanne) is an empathetic, always-on AI assistant hosted on Telegram. This companion bot is designed to listen, comfort, and offer a supportive presence for individuals who may feel isolated or distressed. Powered by OpenRouter AI models, TeleAI aims to alleviate loneliness and prevent mental health crises.

Disclaimer

TeleAI is meant to lend a supportive ear and offer temporary comfort. It is not designed to replace real relationships, physical human touch, or professional mental health counseling.

🌟 Key Features

  • Empathetic Listening: Engineered to provide supportive, non-judgmental conversational outlets.
  • Continuous Chat Memory: Secures and stores conversational context locally per chat.id in data/dataset.json.
  • JSON-Driven Action Routing: Automatically parses structured AI responses to execute modular scripts in real time.
  • Extensible Commands: Includes clear-chat, verse (Bible Gateway scraping), guitar (Ultimate Guitar song search), and imgen (Lumenfall AI image generation).
  • Flexible Setup: Supports production-ready Webhooks (Express) and local development Long Polling.

🛠️ Technology Stack

Technology Role
TypeScript Strict typing for enterprise reliability and self-documenting code.
Node.js & TSX Runtime and direct high-performance execution of TypeScript.
Express 5 Minimalist web framework serving webhooks and client pages.
node-telegram-bot-api Wrapper library interacting with the Telegram Bot API.
Axios Client for communicating with the OpenRouter AI endpoints.

🤖 JSON-Driven Command Routing

Every message from Krysanne is structured as a JSON block, which allows the bot's core loop to run custom backend functions when needed: 

{
  "message": "A human-readable explanation or chat response.",
  "command": "clear-chat | verse | guitar | imgen | new-thread | (empty)",
  "parameter": "Arguments or search query for the command",
  "title": "Optional topic title (e.g., for creating or renaming forum topics)"
}

🚀 Quick Start

1. Prerequisites

  • Node.js (v18+ recommended) and npm.
  • A Telegram Bot Token from @BotFather.
  • An OpenRouter API Key and (optionally) Lumenfall AI key.

2. Installation

# Clone the repository
git clone https://github.com/RyannKim327/Tele-AI
cd TeleAI

# Install dependencies
npm install

3. Environment Setup

Create a .env file in the root directory:

TELEGRAM_TOKEN=your_telegram_bot_token
AI_TOKEN=your_openrouter_api_key
BOT_CODE=your_bot_secret_code
LUMENFALL_API=your_lumenfall_api_key

# Webhook (Optional - falls back to polling if blank)
# WEBHOOK_URL=https://your-domain.com
# PORT=3000

4. Running

# Run in development/production mode
npm start
